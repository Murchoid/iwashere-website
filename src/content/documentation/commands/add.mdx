# add – Save your context

Create a new note describing what you're working on.

## Usage

```bash
iwashere add <message> [options]
```

### Options

| Flag        | Description                             |
|-------------|-----------------------------------------|
| `--session` | Adds the note to the current session    |
| `--tags`    | Comma-separated tags                    |
| `--branch`  | Override detected git branch           |

### Examples

```bash
iwashere add "Implementing JWT authentication"
# Note saved (ID: 4d28a474)

iwashere add "Fix memory leak" --tags bug,performance
```
