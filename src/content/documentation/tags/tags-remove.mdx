# tag remove – Remove tags from a note

Deletes one or more tags from an existing note.

## Usage

```bash
iwashere tag remove <note-id> <tag1>[,<tag2>,...]
```

## Examples

### Remove one tag

```bash
iwashere tag remove 4d28a474 bug
# Tags [bug] removed from note #4d28a474
```

### Multiple tags

```bash
iwashere tag remove 4d28a474 jwt,security
```

### Remove all tags

```bash
iwashere tag remove 4d28a474 --all
# Removed all tags from note 4d28a474
```

> Verify with:

```bash
iwashere show 4d28a474
```
