# Tags

Tags help you organize and filter your notes. Add one or more tags to any note for easy categorization and searching.

## Common Tag Examples

| Tag | Use Case |
|-----|----------|
| `bug` | Bug fixes and issues |
| `feature` | New feature development |
| `docs` | Documentation work |
| `refactor` | Code refactoring |
| `test` | Testing related |
| `urgent` | High priority items |

## Commands

- [tag add](/documentation/tags-add) - Add tags to a note
- [tag remove](/documentation/tags-remove) - Remove tags from a note
- [tag list](/dodocumentationcs/tags-list) - List notes by tag
