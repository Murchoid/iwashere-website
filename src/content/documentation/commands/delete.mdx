# delete - Remove a Note

The `delete` command removes a note permanently.

## Usage

```
iwashere delete <note-id> 
```

### Examples
Delete a note (with confirmation)

```
$ iwashere delete 4d28a474

Note deleted
```

⚠️ Warning
Deletion is permanent. There is no trash or undo. Consider archiving important notes instead.

## Related Commands
- [list](/documentation/list) - Find note IDs

- [show](/documentation/show) - Verify note content before deleting
