# session continue – Resume a paused session

Restart timing for a session that was previously paused. The elapsed time carries forward.

## Usage

```bash
iwashere session continue
```

## Examples

```bash
iwashere session continue
# Session 'Authentication Work' continued
```

```bash
iwashere add "Implement refresh token rotation" --session
```

Status shows the resumed session with updated duration.

### Time tracking

When continued, iwashere:

1. Records the resume timestamp.
2. Restarts the timer.
3. Adds the new active period to the previous total.

Total = previous active periods + current.

### Notes

- Only paused sessions can be continued.
- Ended sessions cannot be resumed.
