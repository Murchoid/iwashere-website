# remind - Set Reminders for Your Notes
The remind command helps you remember to come back to important notes. Never forget to follow up on a task or idea again.

Sometimes you add a note but need to be reminded about it later. Maybe it's:

- A task you need to do tomorrow

- A follow-up for next week's meeting

- Something to check after the weekend

- A reminder to revisit a complex problem

The remind command lets you attach a reminder to any note, and iwashere will notify you when it's due.

## Usage
```bash
$ iwashere remind <note-id> --at <when>

$ iwashere remind list
$ iwashere remind delete <reminder-id>
```

## Options
|Option	|Description|
|--------|-----------|
|--at	|When to remind (e.g., "tomorrow 9am", "in 2 hours", "next Monday")|
|--repeat	|Repeat interval: "daily", "weekly", "monthly"|
|--message	|Custom reminder message (defaults to note message)|
|--notify	|Send system notification when due|

## Time Formats
iwashere understands natural language time expressions:

|Example	|Description|
|----------|-------------|
|--at "tomorrow 9:00"|	Tomorrow at 9:00 AM|
|--at "tomorrow 21:00"|	Tomorrow at 9:00 PM|
|--at "today 21:00"|	Tomorrow at 9:00 PM|
|--at "today 9:00"|	Tomorrow at 9:00 AM|
|--at "in 2h"|	2 hours from now|
|--at "in 30m"|	30 minutes from now|
|--at "Friday 17:00"|	This Friday at 5:00 PM|
|--at "2025-12-31 23:59"|	Specific date and time|
