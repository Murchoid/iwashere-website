# Commands Reference

iwashere provides a set of commands to manage your context. All commands follow the pattern:

```bash
iwashere <command> [arguments] [options]
```

## Core Commands

|Command|	Description|
|-------|--------------|
|init	|Initialize iwashere in current directory|
|add	|Add a new note|
|show	|Show a specific note|
|list	|List all notes|
|edit	|Edit a note|
|delete	|Delete a note|
|branch	|Branch-specific notes|
|Session |Commands|
|Command	|Description|
|session |start	Start a new work session|
|session |pause	Pause current session|
|session |continue	Continue a paused session|
|session |end	End current session|

## Tag Commands

|Command|	Description|
|-------|--------------|
|tag add|	Add tags to a note|
|tag remove|	Remove tags from a note|
|tag list|	List notes by tag|

## Sharing Commands

|Command|	Description|
|-------|--------------|
|share	|Share notes with teammates|
|show-shared	|View notes shared with you|

## Utility Commands

|Command|	Description|
|-------|--------------|
|status	|Show current context|
|version|	Show version information|
|update|	Update to latest version|
|help	|Show help for any command|
