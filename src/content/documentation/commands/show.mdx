# show - View a Note

The `show` command displays a specific note by its ID.

## Usage

```
iwashere show <note-id>
```

## Examples
Show a note
```
$ iwashere show 4d28a474

3m ago (main) - Implementing JWT authentication
```

### Note ID
Note IDs are generated when you add a note. You can find them using `iwashere list` or `iwashere branch`.

## Related Commands
- list - List all notes

- edit - Modify a note
