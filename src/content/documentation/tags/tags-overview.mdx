# Tags

Tags help you organise and search your notes. Add one or more tags to any note; they are stored in lowercase and handled case‑insensitively.

## Common examples

| Tag       | Typical use case           |
|-----------|----------------------------|
| `bug`     | Bug fixes and issues       |
| `feature` | New feature development    |
| `docs`    | Documentation work         |
| `refactor`| Code refactoring           |
| `test`    | Testing related tasks      |
| `urgent`  | High‑priority items        |

## Commands

- [tag add](documentation/tags-add) – add tags to a note
- [tag remove](documentation/tags-remove) – remove tags from a note
- [tag list](documentation/tags-list) – list notes by tag
