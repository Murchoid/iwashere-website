# Installing iwashere

Choose the section corresponding to your operating system.

## Windows

### Quick install (≈ 2 minutes)

1. Download `iwashere-windows-amd64.exe` from the [releases page](https://github.com/Murchoid/iwashere/releases).
2. Rename it to `iwashere.exe`.
3. Create a directory: `C:\tools\iwashere\`.
4. Move the executable there.
5. Add the folder to your `PATH`:
   - Press **Windows Key + X** → System → Advanced system settings → Environment Variables.
   - Under **System variables** edit `Path`, click **New**, and add `C:\tools\iwashere`.
   - Click **OK** to close all dialogs.
6. Restart your terminal.
7. Verify:

```bash
iwashere --help
```

> ⚠️ You must restart or open a new shell for the PATH change to take effect.

### Using Scoop (recommended)

```powershell
scoop bucket add iwashere https://github.com/Murchoid/scoop-iwashere
scoop install iwashere
```

## Linux

### Quick install

```bash
wget https://github.com/Murchoid/iwashere/releases/download/v0.2.10/iwashere-linux-amd64
chmod +x iwashere-linux-amd64
sudo mv iwashere-linux-amd64 /usr/local/bin/iwashere
iwashere --help
```

### Distribution packages

| Distro          | Command |
|-----------------|---------|
| Debian/Ubuntu   | `curl -L ... -o iwashere.deb && sudo dpkg -i iwashere.deb` |
| Fedora/RHEL     | `sudo rpm -ivh https://.../iwashere_0.2.10_linux_amd64.rpm` |
| Any (binary)    | `curl -L ...tar.gz \| tar xz && sudo mv iwashere /usr/local/bin/` |

## macOS

### Homebrew (recommended)

```bash
brew tap Murchoid/homebrew-iwashere
brew install iwashere
```

### Standalone binary (Intel)

```bash
curl -L https://github.com/Murchoid/iwashere/releases/download/v0.2.10/iwashere_0.2.darwin.tar.gz | tar xz
sudo mv iwashere /usr/local/bin/
```

### Standalone binary (Apple Silicon)

```bash
curl -L https://github.com/Murchoid/iwashere/releases/download/v0.2.10/iwashere_0.2.10_darwin_amd64.tar.gz | tar xz
sudo mv iwashere /usr/local/bin/
```
