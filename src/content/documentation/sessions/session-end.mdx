# session end - Complete a Session

The `end` command finishes a work session. Once ended, a session cannot be resumed.

## Usage

```
iwashere session end
```

### Examples
End current session

```
$ iwashere session end
Session 'Authentication Work' ended (total: 3h 15m)
```

View all sessions
```
$ iwashere session list
 Sessions
================

├─  0ce90c9174cea221fb204ec933a6fda6 (Commands documentation) (started 13m ago, paused)

└─  92998a7d94c714393ca474d7a89f90d2 (Setup) (started 3h ago, lasted 2h20m0s)
```

View a specific session
```
$ iwashere session list 123
 Session: Commands documentation
================================

Started:  Mar 10, 2026 at 00:28 (15m ago)
Ended:    Mar 10, 2026 at 00:28 (15m ago)
Duration: 0s
```

What Happens When You End
- Session state becomes ended

- Final total time is calculated

- No more notes can be added

- Session appears in history
