# tag list – List notes by tag

Display notes that contain a given tag or show all tags in use.

## Usage

```bash
iwashere tag list [tag] [options]
```

## Examples

### Notes with a tag

```bash
iwashere tag list bug
```

### All tags in use

```bash
iwashere tag list

All Tags
============
bug (3 notes)
auth (5 notes)
jwt (2 notes)
performance (1 note)
```

### Tag cloud

```bash
iwashere tag list --cloud

bug         ****
auth        *****
jwt         **
performance *
```

### Filters

```bash
iwashere tag list bug --limit 5
iwashere tag list auth --since "this week"
```
