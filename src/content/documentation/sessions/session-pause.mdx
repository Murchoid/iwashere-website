# session pause – Pause the current session

Temporarily stops timing for the active session. Use when you take a break or switch context.

## Usage

```bash
iwashere session pause
```

## Examples

```bash
iwashere session pause
# Session 'Authentication Work' paused (total: 2h 15m)
```

Status will reflect `Session paused`.

### What happens when paused

- Timer stops accumulating.
- You cannot add notes to the paused session.
- Resume with `iwashere session continue`.

### Notes

- Only an ongoing or continued session can be paused.
- Paused sessions do not count toward total time.
