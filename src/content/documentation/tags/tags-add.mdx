# tag add – Add tags to a note

Use the `add` subcommand to attach tags to an existing note.

## Usage

```bash
iwashere tag add <note-id> <tag1>[,<tag2>,...]
```

## Examples

### Single tag

```bash
iwashere tag add 4d28a474 bug
# New tag added to note #4d28a474
```

### Multiple tags

```bash
iwashere tag add 4d28a474 auth,jwt,security
```

### During note creation

```bash
iwashere add "Fix login bug" --tags bug,auth
```

The saved note will show tags when using `iwashere list`.

> Tags are case‑insensitive and stored as lowercase. Duplicates are ignored.
