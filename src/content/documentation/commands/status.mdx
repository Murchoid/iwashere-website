# status - quick info of what you were doing

## Usage

```
iwashere status
```

### Examples

```
$ iwashere status

iwashere status
=================

You were working on 'Route implementation' (1h ago)
Session ongoing
Last note: JJust finished the writing the user controllers, Work on blog controllers
feature

Modified files:
   • src/server.ts (unstaged)
   • src/controllers/userControllers.ts (unstaged)

Related notes from this session:
   └─ 1h ago - Just finished the writing the user controllers, Work on blog controllers [feature]

What's next?
   • Add a note: iwashere add "next task"
   • End session: iwashere session end
   • View all notes: iwashere list

```

### What Status Shows

|Section	|Description|
|-------|-------------|
|**Session**	|Active session name and duration|
|**Last Note**	|Most recent note in current session|
|**Modified Files**	|Files changed (from notes + git)|
|**Related Notes**	|Recent notes in current session|
|**Next Steps**|	Helpful command suggestions|


### With No Active Session

```
$ iwashere status
iwashere status
=================

No active session
   Start one with: iwashere session start "session name"

Last note: Fix memory leak (2 days ago)

 What's next?
   • Start a session
   • View all notes: iwashere list
   ```

### With Long-Running Session Warning

```
$ iwashere status
iwashere status
=================

You were working on 'Database Work' (3 days ago) ⚠️
   This session has been running for 3 days!
   Did you forget to end it?
```
