# Getting started
Welcome to iwashere! This guide will help you set up and start using iwashere in your projects.

## Installation

### Linux (Debian/Ubuntu)
```
curl -L https://github.com/Murchoid/iwashere/releases/latest/download/iwashere_Linux_x86_64.deb -o iwashere.deb
sudo dpkg -i iwashere.deb
```

### macOS
```
brew install Murchoid/tap/iwashere
```

### Windows

```
scoop bucket add iwashere https://github.com/Murchoid/scoop-iwashere
scoop install iwashere
```

## Quick Start
### Initialize iwashere in your project

```
cd your-project
iwashere init
```

### Add your first note

```
iwashere add "Working on authentication feature"
```
### Check your status

```
iwashere status
```

### Basic Commands

- iwashere init	-Initialize a project 
- iwashere add <message>	-Add a note
- iwashere list	-List all notes
- iwashere status	-Show current context
