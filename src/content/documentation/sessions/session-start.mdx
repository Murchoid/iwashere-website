# session start – Begin a work session

Start a new session. All notes added with `--session` will be attached to it until you pause or end it.

## Usage

```bash
iwashere session start <session-name>
```

## Examples

Start:

```bash
iwashere session start "Authentication Work"
# Session 'Authentication Work' started
```

Confirm with status:

```bash
iwashere status
```

```
… (output) …
```

Add notes within session:

```bash
iwashere add "Research JWT libraries" --session
iwashere add "Implement token generation" --session
```

### Notes

- Only one session may be active at a time.
- Starting a session while another is ongoing causes an error; pause or end the existing session first.
- Notes are not added to the current session unless you use `--session`.
