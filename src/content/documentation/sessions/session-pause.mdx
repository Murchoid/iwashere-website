# session pause - Pause Current Session

The `pause` command temporarily stops the current session. Use this when you're interrupted or need to switch context briefly.

## Usage

```
iwashere session pause
```

### Examples
Pause active session

```
$ iwashere session pause
Session 'Authentication Work' paused (total: 2h 15m)
```

Status shows paused session
```
$ iwashere status

iwashere status
=================
You were working on 'Commands documentation' (just now)
Session paused

Last note: Just finished the setup, Work on auth middleware
feature

Modified files:
   • src/middlewares/auth.ts (unstaged)

Related notes from this session:
   └─ 1h ago - Just finished the setup, Work on auth middleware [feature]

What's next?
   • Add a note: iwashere add "next task"
   • End session: iwashere session end
   • View all notes: iwashere list
```

### What happens when paused
1. Timer stops accumulating

2 .Notes cannot be added to paused session

3. Session can be resumed with continue

### Notes
1. Only active (ongoing or continued) sessions can be paused

2. Paused sessions don't accumulate time

3. Use continue to resume

## Related Commands
- session continue - Resume paused session

- session end - End session permanently

- status - See session state
