# Getting Started with iwashere

Welcome to iwashere! This guide will help you get up and running quickly.

## What is iwashere?

iwashere is a context preservation tool for developers. It remembers where you left off in your coding projects, so you never have to ask "What was I working on again?"

## Key Features

- **Context Tracking** - Save notes about your current work
- **Git Integration** - Automatically captures branch and commit context
- **Sessions** - Group related notes into work sessions
- **Tags** - Organize notes with tags
- **Team Sharing** - Share notes securely with teammates
- **Self-Updating** - One command to stay current
- **Reminders** - Remind yourself of tasks you dont want to forget
