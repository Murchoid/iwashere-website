# branch – Branch-specific notes

Display notes associated with a git branch. By default it shows the current branch.

## Usage

```bash
iwashere branch [branch-name] [options]
```

### Options

| Flag     | Description          |
|----------|----------------------|
| `--short`| Show compact notes   |

## Examples

### Current branch

```bash
$ git branch
* feature/auth
  main

$ iwashere branch
Git context: feature/auth @ 4cc7baa
You have uncommitted changes
[51m ago] (feature/auth) 613f9bb492427164a545e7c851c63215: Just finished the setup, Work on the commands mdxs 
  #feature
```

### Specific branch

```bash
iwashere branch main
```

### Show all branches

Use `iwashere list` instead.
