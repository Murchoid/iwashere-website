# list – List all notes

Shows notes from all branches (or use `branch` for branch-specific output).

## Usage

```bash
iwashere list [options]
```

### Options

| Flag       | Description                    |
|------------|--------------------------------|
| `--limit N`| Show only N most recent notes  |
| `--short`  | Compact output format          |

### Examples

```bash
# All notes
iwashere list

# Limit to 3 most recent
iwashere list --limit 3

# Compact view
iwashere list --short
```

#### Sample outputs

```bash
#feature
[2 hours ago] (feature/auth) 4d28a474: Implementing JWT authentication

#bug #performance
[yesterday] (main) 8f3e2a1b: Fix memory leak in connection pool

[3 days ago] (feature/ui) 7b5c3d1a: Update README
```

```bash
2h ago (feature/auth) Implementing JWT authentication
1d ago  Fix memory leak
3d ago  Update README
```
