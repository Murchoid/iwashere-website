# session end – Complete a session

Marks the current session as finished. You cannot resume it afterwards.

## Usage

```bash
iwashere session end
```

## Examples

```bash
iwashere session end
# Session 'Authentication Work' ended (total: 3h 15m)
```

### Viewing sessions

```bash
iwashere session list

Sessions
================
…
```

```bash
iwashere session list 123

Session: Commands documentation
…
```

### What happens when you end a session

- State becomes `ended`.
- Final total time is computed.
- No more notes may be added.
- The session remains in history.
