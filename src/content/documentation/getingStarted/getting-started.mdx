# Getting Started with iwashere

Welcome! **iwashere** is a CLI tool that helps you keep track of what you're working on so you can pick up where you left off.

## What is iwashere?

- **Context tracking** – Save notes about your current work.
- **Git integration** – Captures branch & commit context automatically.
- **Sessions** – Bundle related notes into work blocks.
- **Tags** – Label and filter notes.
- **Team sharing** – Securely send notes to teammates.
- **Self‑updating** – One command to stay current.
- **Reminders** – Avoid forgetting tasks.
