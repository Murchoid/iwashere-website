# delete – Remove a note

Permanently deletes a note from your history.

## Usage

```bash
iwashere delete <note-id>
```

## Example

```bash
iwashere delete 4d28a474
# Note deleted
```

> ⚠️ Deletion is permanent. There is no undo or trash. Consider archiving important notes instead.
