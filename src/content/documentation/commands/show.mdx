# show – View a note

Display a specific note by its ID.

## Usage

```bash
iwashere show <note-id>
```

### Example

```bash
$ iwashere show 4d28a474
3m ago (main) - Implementing JWT authentication
```

#### Note IDs

Generated when you add a note. Use `iwashere list` or `iwashere branch` to find them.
